# zuki.dev

Repository for the [zuki.dev](https://zuki.dev)

## Requirements

- [aqua](https://github.com/aquaproj/aqua)

## Setup

```bash
# Clone this repo
git clone --recursive git@github.com:corrupt952/zuki.dev

# Install tools via aqua
aqua i
```

## Run local server

```bash
# Run hugo server
hugo server
```
